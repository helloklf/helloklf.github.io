
## 分组
- 使用`group`节点分组你的功能节点
- 使界面显示更有层次感
- 在`group`节点，可以根据需要设置`title`和`visible`属性

```xml
<?xml version="1.0" encoding="UTF-8" ?>
<page>
    <group title="分组标题">
        <action>
            <!-- ... 此处省略 action 的详细定义 -->
        </action>
        <switch>
            <!-- ... 此处省略 switch 的详细定义 -->
        </switch>
    </group>
</page>
```
